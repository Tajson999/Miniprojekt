    • Kort fakta om CSS

    CSS står för Cascading Style Sheets och är ett språk som som används för att beskriver presentationsstilen för ett strukurerat dokument. Medan HTML bara säger åt en var allt (headers, titles) ska ligga, säger CSS till hur den kommer se ut. Vilket typsnitt den ska ha, om den ska ha en backgrundsfärg m.m. Den formar hur sidan kommer att se ut.


    Syntax består av:

    p {font-size:16px;}
